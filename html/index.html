流光计划

简述

流光计划是一个有关于 2D 游戏引擎的项目。

目标是制作一个泛用型的引擎程序，为有编程基础的人提供一个充满可能性的平台，超越并取代RPGMaker等游戏制作软件。

项目名称：流光(flux) 引擎代码：C++ 开发环境：VS2008, windows 游戏代码：lua 开发环境：任意文本编辑器 当前版本：0.07

基本功能

输入控制
声音支持
物理引擎
ttf 字体渲染
加载所有常用图片格式
lua 脚本支持

特性

  • 存在镜头系统，支持锁定某个单位
  • 开发者对游戏有极大地控制权
  • 控件动画，提供对移动、旋转、改变透明度、改变颜色、改变大小等效果的支持，同时可以在动画结束后加上回调函数
  • 控件动画列表，可以在支持的动画效果中进行任意的组合，以实现一系列的变换，也同时支持循环播放。
